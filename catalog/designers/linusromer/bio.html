Linus Romer is a type designer and font developer specializing in font technology and open-source tools for the type design community. He has developed several widely-used utilities for font engineering, including tools for curve manipulation, METAFONT conversion, and font interpolation.

His work focuses on technical aspects of type design, creating plugins for FontForge and developing software that bridges traditional METAFONT technology with modern outline font formats. He is recognized as a contributor to the Arctic Code Vault and has an active presence in the open-source typography community.
