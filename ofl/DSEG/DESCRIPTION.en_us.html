DSEG is a free font which imitates an LCD Display (7SEG, 14SEG, Weather icons etc.) DSEG is designed by keshikan (https://www.keshikan.net) DSEG has special features: - includes the roman-alphabet and symbol glyphs. - many types (over 50) are available.

For more information, visit DSEG support page. - English: https://www.keshikan.net/fonts-e.html - Japanese: https://www.keshikan.net/fonts.html
