Alumni Sans Collegiate One is a stand-alone font based on Alumni Sans, which is a variable font containing Roman and Italic styles. This version is a display cut of the ExtraBold which adds a decorative outline. Use this variation for situations requiring a sporty look.

Collegiate One is designed to be used as a display font above 32pt in print (assuming 300 dpi) and above 72px in digital media.

It comes with Latin Character sets including Western, Central, and Vietnamese language support.

To contribute, see github.com/googlefonts/alumni-sans-collegiate.
