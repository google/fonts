Bowlby One is the lowercase version of Bowlby One SC. It has been designed for use as a display font.

The form is a fusion of a handful of designs scanned from old, early Twentieth Century type specimens. Bowlby One is perfect for big bold headlines and display uses that need a slightly roughened look. Bowlby is a web font, designed to be used freely across the internet by web browsers on desktop computers, laptops, cloud systems and mobile devices.
