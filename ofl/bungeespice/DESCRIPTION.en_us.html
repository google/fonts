Bungee Spice is a derivative of David Jonathan Ross font Bungee. Bungee Spice is a COLRv1 font exhibiting multiple gradients within each glyph.

Bungee Spice is released under the SIL Open Font License.

David Jonathan Ross superfamily Bungee includes:

  • Bungee
  • Bungee Hairline
  • Bungee Inline
  • Bungee Outline
  • Bungee Shade

The Bungee project is led by David Jonathan Ross, a type designer based in the USA. To contribute, see github.com/djrrb/Bungee
