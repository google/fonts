FONTLOG for the Chivo fonts

This file provides detailed information on the Chivo Font Software.
This information should be distributed along with the Chivo fonts
and any derivative works.

Basic Font Information

Chivo is a typeface family that supports Unicode language range: Latin-1

To contribute to the project contact Omnibus-Type at omnibus.type@gmail.com
