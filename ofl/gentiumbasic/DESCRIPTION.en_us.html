The Gentium Basic font family is based on the original Gentium design, but with additional weights. The family comes with a complete regular, bold, italic and bold italic set of fonts.

The supported character set, however, is much smaller than for the main Gentium Plus fonts. These "Basic" fonts support only the Basic Latin and Latin-1 Supplement Unicode ranges, plus a selection of the more commonly used extended Latin characters, with miscellaneous diacritical marks, symbols and punctuation. Please see the Gentium project homepage for more details on languages supported.

The Gentium Book Basic family is very similar but has a slightly darker weight.
