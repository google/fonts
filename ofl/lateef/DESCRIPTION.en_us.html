Lateef, an extended Arabic font, is named after Shah Abdul Lateef Bhitai, the famous Sindhi mystic and poet. Lateef has a tall x-height which gives the font an elegant feel. It is intended to be an appropriate style for use in Sindhi and other languages of the South Asian region.

This font was developed by SIL, and you can learn more about it at scripts.sil.org/Lateef
