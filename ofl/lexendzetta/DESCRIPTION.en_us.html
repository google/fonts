Lexend is a collection of seven font families empirically shown to significantly improve reading-proficiency. As prescription eyeglasses achieve proficiency for persons with short-sightedness, Lexend's families were developed using Shaver-Troup Formulations.

We will eventually release all seven families as a single variable font featuring its own custom axis.

To contribute, see github.com/ThomasJockin/lexend. For more information, see thomasjockin.github.io/lexend/.
