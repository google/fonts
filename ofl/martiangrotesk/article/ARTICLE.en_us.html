Martian Grotesk is a sans-serif variable typeface family with a distinctive personality. It ships as a single variable font with two axes—weight (Thin to UltraBlack) and width (Condensed to UltraExpanded)—and a subtly brutalist character that stands out visually, while the body-text weights are finely tuned for paragraph use.

The typeface is especially optimized for on-screen use. Vertical metrics are balanced so that text labels appear vertically centered within UI elements and align well with icons. Character heights are tuned to the pixel grid at common UI sizes (10, 15, 20 pixels, and so on) for extra clarity, and both PostScript and TrueType hinting are applied automatically.

Stem widths are kept constant within each weight, so shrinking or stretching letters across the width axis preserves visual harmony. Distinctive details—overhanging terminals in a, e, and s, monospaced-style hooks in f, j, and t—give the text a recognizable flavor without sacrificing readability.

The typeface supports 308 languages: 297 based on the Latin script and 11 using the Cyrillic script.

Martian Grotesk is the proportional sibling of Martian Mono. To contribute, see github.com/evilmartians/grotesk.
