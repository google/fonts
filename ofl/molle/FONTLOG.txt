FONTLOG for the Molle font

This file provides detailed information on Molle Font Software. 
This information should be distributed along with the Molle 
font and any derivative works.

Basic Font Information

Molle is a Unicode typeface family that supports languages 
that use the Latin script and its variants, and could be expanded 
to support other scripts.

More specifically, this release partially supports the following Unicode 
ranges: Latin-1

To contribute to the project contact Eben Sorkin at sorkineben@gmail.com

There are four Source files:

1. Molle.vfb Original Source Files with 
   contour overlaps

2. Molle-OTF.sfd Merged contours and 
   optimized file corresponding to the OTF file

3. Molle-TTF.sfd TrueType outlines with 
   hinting adjustments corresponding to the TTF file

4. Molle-OTF Merged contours and 
   optimized Opentype-CFF file
