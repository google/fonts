## Summary

Noto Sans Syriac was sourced from the notofonts/syriac repository. The family was published as a variable font with a `wght` axis (100–900), covering the Syriac script (Syrc) and supporting Assyrian Neo-Aramaic, Classical Syriac, Syriac, Turoyo, and Arabic written in Syriac script. The notofonts/syriac repository also served as the source for the Syriac Eastern and Syriac Western variants. Files were built using the gftools-builder pipeline with `sources/config-sans-syriac.yaml`. The font was added to Google Fonts on 2020-11-19 and released at version 3.000.
