# Overlock SC — Source Metadata Investigation

**Model**: Claude Opus 4.6
**Date**: 2026-03-12

## Summary

Overlock SC is the small-caps companion family to Overlock, designed by Dario Manuel Muhafara of TIPO digital typefoundry (Buenos Aires, Argentina). Both families were added to Google Fonts on 2011-12-19 and share the same upstream source.

## Designer

Dario Manuel Muhafara operates the TIPO digital typefoundry at tipo.net.ar. See the Overlock investigation for full details on the designer search.

## Upstream Repository Search

Overlock SC shares its upstream with Overlock. The same `librefonts/overlock` mirror repository was investigated. No canonical designer-owned repository was found for either Overlock or Overlock SC.

The `librefonts/overlock` repository (https://github.com/librefonts/overlock) is a librefonts organization mirror, not the designer's own repository. The `src/` folder contains VFB and SFD sources only — no UFO or Glyphs files.

No GitHub account for Dario Manuel Muhafara was found on GitHub.

## Conclusion

No canonical designer-owned upstream repository was found for Overlock SC. The shared upstream with Overlock provides only VFB and SFD sources via a librefonts mirror. No METADATA.pb source block was added.
