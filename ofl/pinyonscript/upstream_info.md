## Font Description

Pinyon Script is an elegant, formal calligraphic script typeface originally designed by Nicole Fally and added to Google Fonts in 2011. It was later taken over and maintained by Sorkin Type. The font features fine, high-contrast strokes characteristic of copperplate engraving styles. It is classified as a display handwriting font and covers Latin and Latin Extended scripts, as well as Vietnamese. The copyright dates to 2022, reflecting a recent update to the font files.

## Designer
