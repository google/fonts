## Summary

Playwrite ID (display name: Playwrite Indonesia) was designed by TypeTogether (Veronika Burian, José Scaglione) and published under the OFL. It was sourced from the TypeTogether/Playwrite GitHub repository, a monorepo containing all Playwrite regional variants. The family provided a variable font in the Regular weight axis representing the Indonesian handwriting style. Files were sourced from `fonts/variable/` in the repository. The font was added to Google Fonts on 2024-05-16.
