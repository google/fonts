## Summary

Playwrite IT Moderna Guides (display name: Playwrite Italia Moderna Guides) was designed by TypeTogether (Veronika Burian, José Scaglione) and published under the OFL. It was sourced from the TypeTogether/Playwrite GitHub repository, a monorepo containing all Playwrite regional variants. The family provided a static Regular font with handwriting guide overlays, along with documentation and article files for the Italia Moderna style. Files were sourced from `fonts/ttf/` and `documentation/about-guides/IT_Moderna/` in the repository. The font was added to Google Fonts on 2024-11-27.
