Rubik Beastly is an original, script-generated design based on the GoogleFont Rubik by Hubert & Fischer, Meir Sadan and Cyreal. The code used to generate it can be found at github.com/NaN-xyz/Glyph-Filters.xyz.

Check out generativefonts.xyz for more info.

To contribute, see github.com/NaN-xyz/Rubik-Filtered.
