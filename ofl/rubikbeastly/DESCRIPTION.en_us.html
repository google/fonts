Rubik Beastly is an original, script-generated design based on the Google Fonts Rubik by Hubert and Fischer, Meir Sadan and Cyreal. The code used to generate it can be found at github.com/NaN-xyz/Glyph-Filters.

Check out generativefonts.xyz for more info.

To contribute, see github.com/NaN-xyz/Rubik-Filtered.
