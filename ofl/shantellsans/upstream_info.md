## Summary

Shantell Sans was designed by Shantell Martin in collaboration with Arrow Type and Anya Danilova. The font was built as a variable font with four custom axes: BNCE (Bounce, -100 to 100), INFM (Informality, 0 to 100), SPAC (Spacing, 0 to 100), and wght (Weight, 300 to 800). Both upright and italic variable fonts were included. The family supported Latin, Latin Extended, Cyrillic, Cyrillic Extended, and Vietnamese subsets. The project had a dedicated minisite at shantellsans.com.
