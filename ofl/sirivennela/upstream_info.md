## Summary

Sirivennela was designed by Appaji Ambarisha Darbha and maintained in a personal GitHub repository. The family consisted of a single Regular weight static TTF targeting the Telugu script (primary script: Telu). It also supported the Latin subset. The build configuration was located at `sources/config.yaml`.
