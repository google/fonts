# Slabo 27px — Upstream Source Investigation

## Summary

Slabo 27px is a size-specific slab-serif typeface optimized for 27px on-screen rendering, designed by John Hudson at Tiro Typeworks Ltd (www.tiro.com) and released in 2013. It was added to Google Fonts on 2014-05-30. It is a sibling typeface to Slabo 13px, sharing the same upstream repository.

## Designer

John Hudson — Tiro Typeworks Ltd (tiro.com)

## Repository Search

A search for canonical upstream repositories was conducted on GitHub. The following repository was found:

- **TiroTypeworks/Slabo** (https://github.com/TiroTypeworks/Slabo) — repository owned by Tiro Typeworks, containing UFO sources for both the 13px and 27px variants. The repository contained:
  - `UFOs/Slabo 27px b002.ufo` — UFO source for the 27px variant
  - `UFOs/Slabo 13px b004.ufo` — UFO source for the 13px sibling variant
  - `FontLab Sources/` — FontLab source files
  - `TTFs/` — compiled fonts

The repository was confirmed as non-archived with an active `master` branch.
