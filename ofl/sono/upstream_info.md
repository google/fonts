## Summary

Sono was designed by Tyler Finck and maintained in a personal GitHub repository. The family was published as a single variable font with two axes: MONO (Monospace, 0 to 1) and wght (Weight, 200 to 800). The default registry value for the MONO axis was set to 1.0, making the monospaced presentation the default. It supported Latin, Latin Extended, and Vietnamese subsets. The build configuration was at `sources/config.yaml`.
