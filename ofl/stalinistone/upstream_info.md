## Research

Stalinist One was designed collaboratively by Alexey Maslov and Jovanny Lemonad
(lemonad@jovanny.ru). The font was released in 2009-2011 and updated in 2015.
Searches on GitHub for repositories by these designers or under the font name
returned no results. No GitHub accounts for Alexey Maslov or Jovanny Lemonad
(lemonad@jovanny.ru) were found to have public font source repositories.


## Notes

The designer listed in METADATA.pb is Alexey Maslov, Jovanny Lemonad. Stalinist
One is described as a post-apocalyptic display typeface designed in Moscow. It
supports Cyrillic, Latin, and Latin Extended character sets and is licensed
under the SIL Open Font License.
