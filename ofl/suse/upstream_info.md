# SUSE — Upstream Source Investigation

**Family**: SUSE
**Designer**: René Bieder
**Date Added**: 2024-08-14
**Category**: Sans-Serif
**Model**: Claude Opus 4.6

## Source Repository

The upstream source was located at:

- **Repository**: https://github.com/SUSE/suse-font
- **Commit**: `7159afb2555b06d3a5c2f90e4d324e59e69c277d`
- **Branch**: main
- **Archive**: https://github.com/SUSE/suse-font/releases/download/v2.001/suse-font-v2.001.zip

## Font Files

The family was published as two variable fonts covering the weight (`wght`) axis:

- `SUSE[wght].ttf` — upright
- `SUSE-Italic[wght].ttf` — italic

## Subsets

The family covered the following subsets: latin, latin-ext, vietnamese.

## Notes

SUSE was designed by René Bieder and published to Google Fonts on 2024-08-14. The font was developed by SUSE, the open-source software company, and hosted under their GitHub organization. Fonts were sourced from release archive v2.001. The same upstream repository also contained the SUSE Mono companion family.
