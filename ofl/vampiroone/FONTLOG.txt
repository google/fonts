FONTLOG for the Vampiro One font

This file provides detailed information on Vampiro Font Software. This
information should be distributed along with the Vampiro font and any
derivative works.

Basic Font Information

Vampiro One is a low contrast script font. It was inspired by the 20th century
Italian tradition of monoline scripts. Vampiro One is best used for display
purposes at medium to large sizes.

Vampiro is a Unicode typeface family that supports languages that use the
Latin script and its variants, and could be expanded to support other scripts.

More specifically, this release partially supports the following Unicode
ranges: Latin-1

There are four source files:

1. VampiroOne-Regular.vfb Original Source Files with 
   contour overlaps

2. VampiroOne-Regular_otf.sfd Merged contours and 
   optimized file corresponding to the OTF file

3. VampiroOne-Regular_ttf.sfd TrueType outlines with 
   hinting adjustments corresponding to the TTF file

4. VampiroOne-Regular.otf PostScript outlines OTF-CFF font.
