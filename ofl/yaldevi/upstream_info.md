## Summary

Yaldevi was designed by Mooniak as a variable sans-serif typeface supporting Latin and Sinhala scripts. The font was sourced from the `mooniak/yaldevi-font` repository at the commit listed above. The variable font file `fonts/variable/Yaldevi[wght].ttf` was copied to the Google Fonts distribution. The weight axis spans from 200 (ExtraLight) to 700 (Bold).

**Model**: Claude Opus 4.6
