## Summary

Yanone Kaffeesatz was designed by Yanone (with contributions by Cyreal) as a variable sans-serif display typeface supporting Latin, Cyrillic, Vietnamese, Math, and Symbols subsets. The font was sourced from the `yanone/kaffeesatz` repository at the commit listed above. The variable font file `fonts/variable/YanoneKaffeesatz[wght].ttf` was copied to the Google Fonts distribution. The weight axis spans from 200 (ExtraLight) to 700 (Bold).

**Model**: Claude Opus 4.6
