## Summary

Yarndings 12 was designed by Sarah Cadigan-Fried as a display symbols typeface inspired by yarn crafts. The font was sourced from the `scfried/soft-type-yarndings` repository at the commit listed above. The static TTF file `fonts/ttf/Yarndings12-Regular.ttf` was copied to the Google Fonts distribution. The font supports Latin, Math, and Symbols subsets.

**Model**: Claude Opus 4.6
