## Summary

Yarndings 20 Charted was designed by Sarah Cadigan-Fried as a display symbols typeface inspired by yarn crafts, featuring charted versions of the 20-stitch grid symbols. The font was sourced from the `scfried/soft-type-yarndings` repository at the commit listed above (shared with Yarndings 12, 12 Charted, and 20). The static TTF file `fonts/ttf/Yarndings20Charted-Regular.ttf` was copied to the Google Fonts distribution. The font supports Latin, Math, and Symbols subsets.

**Model**: Claude Opus 4.6
