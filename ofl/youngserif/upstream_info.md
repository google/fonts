## Summary

Young Serif was designed by Bastien Sozeau as a static serif typeface supporting Latin and Latin Extended subsets. The font was sourced from the `noirblancrouge/YoungSerif` repository at the commit listed above. The static TTF file `fonts/ttf/YoungSerif-Regular.ttf` was copied to the Google Fonts distribution.

**Model**: Claude Opus 4.6
