## Summary

Zen Dots was designed by Yoshimichi Ohira as a static sans-serif display typeface supporting Latin and Latin Extended subsets. The font was sourced from the `googlefonts/zen-dots` repository at the commit listed above. The static TTF file `fonts/ttf/ZenDots-Regular.ttf` was copied to the Google Fonts distribution.

**Model**: Claude Opus 4.6
