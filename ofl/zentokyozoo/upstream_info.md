## Summary

Zen Tokyo Zoo was designed by Yoshimichi Ohira as a static display typeface supporting Latin and Latin Extended subsets. The font was sourced from the `googlefonts/zen-tokyo-zoo` repository at the commit listed above. The static TTF file `fonts/ttf/ZenTokyoZoo-Regular.ttf` was copied to the Google Fonts distribution.
